You mission is to sort list by frequency of numbers which this list includes. If few numbers has an equal frequency - it should be sorted in the according to their natural order. For example: [5, 2, 4, 1, 1, 1, 3] ==> [1, 1, 1, 2, 3, 4, 5]

Input: Chaotic list of numbers.

Output: List of numbers, where numbers sorted by their frequency.

Example:

{% if interpreter.slug == "js-node" %} 
frequencySorting([5, 3, 8, 11, 5, 6, 6, 5]) == [5, 5, 5, 6, 6, 3, 8, 11]
{% else %} 
frequency_sorting([5, 3, 8, 11, 5, 6, 6, 5]) == [5, 5, 5, 6, 6, 3, 8, 11]
{% endif %}

How it is used: For analyzing data using mathematical statistics and mathematical analysis, and for finding trends and predict future changes (systems, phenomena, etc.)

Precondition:
array length <= 100
max number <= 100
